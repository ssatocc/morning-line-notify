# morning-line-notify
Send notifications every day at 6am via LINE using GitHub Actions

## Setup

```bash
cp -p .env.example .env
```

```bash
vi .env
```

```bash
. .env
```

## Send a Message

```bash
bash send.sh $ACCESS_TOKEN
```
